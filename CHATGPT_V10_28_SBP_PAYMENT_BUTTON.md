# v10.28 — отдельная кнопка оплаты по СБП

Добавлено:
- отдельная кнопка “Оплатить по СБП” в форме оплаты;
- компактный визуальный логотип СБП;
- сумма остатка подставляется прямо в кнопку;
- если бонусы и баланс полностью закрывают платеж, кнопка СБП становится неактивной;
- при нажатии показывается прототипный статус перехода к оплате по СБП.

Реальная интеграция СБП не подключалась — это UX-прототип.
