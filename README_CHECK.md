# Как проверять v12

1. Распаковать архив.
2. Открыть `lk_client_portal_preview.html` в браузере.
3. Сделать Ctrl+F5.
4. Проверить экраны:
   - Главная;
   - Счета и оплаты;
   - Форма оплаты;
   - Акты и документы;
   - Заказы и отгрузки;
   - Обращения;
   - Проекты;
   - Базы 1С;
   - мобильное меню;
   - AnyDesk/RuDesktop.

## Что должно сохраниться

- Кнопка оплаты у счета открывает форму.
- Сумма счета подставляется из выбранного документа.
- СБП остается отдельной кнопкой.
- Вымышленные адреса AnyDesk/RuDesktop не показываются.
- Если реальные данные удаленного доступа не переданы в `window.lkRemoteAccess`, показывается только скачивание/запуск по текущей настройке без адреса.
