# v12 Figma UI Merge

Источник логики: `lk_clickable_v11_8_remote_real_device_fix.zip`.

Источник визуального направления: Figma export `Улучшение HTML прототипа.zip`.

## Что перенесено из Figma

- Более чистый B2B SaaS-внешний вид.
- Контрастная навигация слева.
- Светлый фон основного контента.
- Белые карточки с тонкой рамкой и мягкой тенью.
- Более аккуратная типографика.
- Более ровные кнопки, таблицы, поля и статусы.
- Улучшенная визуальная подача формы оплаты.
- Мобильная адаптация модального окна оплаты и карточек.

## Что специально не менялось

- Логика оплаты из v11.8.
- СБП-кнопка и иконка.
- Бонусы и баланс.
- Документы, счета, заказы, обращения, проекты, базы 1С.
- Логика AnyDesk/RuDesktop: не подставлять вымышленные адреса.
- Если реальные данные удаленного доступа не пришли от агента/API — адрес не показывается.

## Важно для production

Это HTML-preview для визуального согласования. Для production нужно переносить визуальную систему в React/Vite source, а не встраивать этот HTML как финальный фронтенд.
