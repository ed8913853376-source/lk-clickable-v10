# Личный кабинет клиента — рабочий пакет v10.25

Это текущий кликабельный прототип личного кабинета для согласования логики, форм, навигации и сценариев.

## Как открыть

1. Распаковать архив.
2. Открыть файл `index.html` в браузере.
3. Для проверки мобильной версии открыть DevTools → режим телефона или открыть файл на смартфоне.

## Что уже есть

- Главная страница ЛК.
- Левое меню с адаптацией под мобильные устройства.
- Разделы: контрагенты, обращения, проекты, договоры, счета и оплаты, акты и документы, заказы и отгрузки, каталог услуг и товаров, базы 1С, лицензии 1С, пользователи, база знаний.
- Кликабельные формы и модальные окна.
- Прототипная логика оплаты, бонусов, баланса, выбора основания платежа.
- Прототипные действия по документам, заказам, обращениям, базам 1С и пользователям.

## Важно

Это не финальная боевая версия сайта и не интеграция с 1С/Битрикс. Это рабочий прототип и пакет для передачи программистам.

Для запуска в реальную работу программистам нужно подключить:

- авторизацию;
- роли и права;
- данные из 1С / CRM;
- создание заявок;
- счета, оплаты, акты, договоры;
- товары, остатки, заказы, отгрузки;
- реальные файлы для скачивания;
- backend/API;
- безопасность и журналирование действий.
