# Черновик интеграций

## Источник данных

Основной источник данных: 1С / CRM.

## Сущности

- Пользователь.
- Контрагент.
- Договор.
- Счет.
- Платеж.
- Акт / УПД / акт сверки.
- Заказ.
- Отгрузка.
- Товар.
- Услуга.
- Обращение.
- Проект / сделка.
- База 1С.
- Конфигурация 1С.
- Лицензии 1С.
- Сервер / RDP / ресурс.

## Действия из ЛК в 1С/CRM

- Создать обращение.
- Добавить комментарий к обращению.
- Прикрепить файл.
- Запросить счет.
- Оплатить счет.
- Заказать акт сверки.
- Скачать документ.
- Заказать товар.
- Заказать услугу.
- Повторить заказ.
- Изменить адрес доставки.
- Запросить изменение лицензий.
- Заказать базу 1С.
- Добавить пользователя.
- Изменить права пользователя.

## Что нужно уточнить у разработчиков 1С

- Какие справочники и документы уже есть.
- Какие API доступны сейчас.
- Какие события нужно отправлять из ЛК в 1С.
- Какие статусы заказов, обращений и оплат использовать.
- Где хранятся файлы документов.
- Как получать остатки товаров и цены.
- Как формировать счет, акт сверки, УПД.
