# План внедрения личного кабинета

## Этап 1. Зафиксировать UX и бизнес-логику

- Проверить все разделы прототипа.
- Утвердить формы и поля.
- Утвердить сценарии оплаты, документов, заказов, обращений и 1С.
- Утвердить мобильную версию.

## Этап 2. Подготовить frontend

- Перенести прототип в нормальную структуру компонентов.
- Разделить страницы, модальные окна, карточки, таблицы и формы.
- Настроить адаптивную верстку.
- Подготовить состояния загрузки, пустых данных и ошибок.

## Этап 3. Backend и интеграция

- Определить API между ЛК, 1С-Битрикс, 1С/CRM и AI Core.
- Настроить авторизацию клиента.
- Настроить роли: руководитель, бухгалтер, сотрудник, технический пользователь, администратор.
- Реализовать получение данных из 1С.
- Реализовать создание заявок, заказов, счетов, запросов документов.

## Этап 4. Тестирование

- Desktop.
- Mobile.
- Разные роли.
- Несколько контрагентов.
- Проверка прав доступа.
- Проверка платежей и документов.

## Этап 5. Внедрение

- Публикация в тестовом окружении.
- Проверка с реальными пользователями.
- Доработки.
- Публикация в боевое окружение.
