# v10.9 — исправление формы “Создать обращение”

Что изменено:

1. Телефон текущего пользователя теперь подставляется по умолчанию.
2. Можно добавить несколько телефонов для связи.
3. Связанный ресурс / объект не заполняется автоматически — по умолчанию стоит “Не выбран”.
4. Кнопка прикрепления файла перенесена вправо от поля “Описание / комментарий”.
5. После выбора файла показывается имя прикрепленного файла.
6. Реальная отправка файлов, создание заявки в 1С/CRM и хранение телефонов не подключены — это прототипная логика.
