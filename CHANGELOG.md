# CHANGELOG v10

## Изменено
- Универсальная форма действий заменена на каталог бизнес-форм v10.
- Добавлены отдельные модальные формы для контрагентов, пользователей, обращений, баз 1С, тонкого клиента, лицензий, серверов/RDP, услуг, товаров, КП, заказов, финансов, документов, проектов и удаленного доступа.
- Быстрые действия на главной открывают формы по смыслу, а не случайные универсальные поля.
- Кнопки AnyDesk и RuDesktop открывают модальное окно `Удаленный доступ`.
- RDP-клиент с настройками открывает профильное окно RDP, без `alert`.
- Базы 1С отделены от серверных ресурсов: в форме базы нет CPU/RAM/SSD/IP.
- Финансовые действия разделены на карту, счет, карту к привязке и обещанный платеж.
- После отправки форм показывается прототипный статус.
- Главная оставлена как рабочая сводка без рекламных баннеров.
- Левое меню остается увеличенным и читаемым, визуальный скроллбар скрыт.
- Обновлены ACTION_MAP, FORMS_SPEC, UX_FLOWS, TODO_QUESTIONS, BUILD_NOTES.

## Ограничения
- Боевые интеграции с 1С, Битрикс, оплатой и скачиванием файлов не подключались.
- Vite build требует восстановления Windows optional binaries Rollup на машине с интернетом.
