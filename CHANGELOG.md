# CHANGELOG

## v10.1 — исправления ChatGPT

- Исправлена логика формы контрагента: кнопка в разделе контрагентов открывает форму реквизитов организации, без полей базы 1С/сервера/товара.
- Разделены бонусы и сертификаты: добавлена простая форма “Сертификат / промокод” с одной строкой ввода кода и информационным баннером.
- Активированы действия по коммерческим предложениям: “Открыть КП”, “Согласовать КП”, “Задать вопрос по КП”, “Скачать КП PDF”.
- В карточку базы 1С добавлен блок “Доступ текущего пользователя”: логин, пароль, показать/скрыть, копировать.
- Уведомления теперь раскрываются как чат/лента: можно выбрать уведомление, посмотреть историю и написать ответ.
- Раздел пользователей переработан: кликабельная таблица, карточка пользователя, логин, пароль, повтор пароля, проверка совпадения, генерация и копирование пароля.
- Карточки базы знаний стали кликабельными: открывается статья, можно скопировать ссылку, создать обращение.
- Добавлено мобильное бургер-меню: на телефонах и планшетах сайдбар открывается кнопкой и закрывается по клику вне меню.
- Исправлена сборка: удалён BOM из package.json и убран отсутствующий импорт tw-animate-css.

## Проверка

`npm install` — выполнено.
`npm run build` — успешно.

## v10.3

- Добавлено плавное сворачивание двух верхних блоков левого меню при прокрутке навигации.
- В блоке клиента в компактном состоянии остается только название клиента.
- В блоке оплаты в компактном состоянии остается только кнопка с суммой внутри.
- Навигация прокручивается отдельно от верхних блоков сайдбара.

## v10.4 — исправление сжатия левой колонки
- Исправлена логика сжатия верхних двух блоков в левой колонке.
- Теперь сжатие запускается от колесика мыши на всей левой панели, а не только от внутреннего scroll-события навигации.
- При прокрутке вниз сначала плавно уменьшаются карточка клиента и блок баланса, затем прокручивается меню.
- При прокрутке вверх в начале меню карточки плавно раскрываются обратно.
- Добавлен fallback: при прокрутке основной страницы левая колонка тоже переходит в компактный режим.
