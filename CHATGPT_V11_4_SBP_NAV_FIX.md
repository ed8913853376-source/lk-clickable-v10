# v11.4 — СБП и форма оплаты из навигации

Исправлено:

1. В форму оплаты добавлен реальный визуальный знак СБП из предоставленного изображения.
2. Кнопка «Пополнить / оплатить» в левой навигации теперь открывает ту же стабильную форму оплаты, где есть отдельная кнопка СБП.
3. На мобильной версии пункт «Счета и оплаты» в burger-меню сразу открывает форму оплаты.
4. Старая логика оплаты по счетам сохранена: кнопки «Оплатить» у счетов открывают форму с суммой и основанием.

Примечание: для прототипа сумма пополнения по умолчанию — 10 000 ₽, ее можно редактировать в форме.
