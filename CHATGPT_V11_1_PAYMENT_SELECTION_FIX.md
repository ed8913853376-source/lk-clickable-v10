# v11.1 — исправление оплаты и выделения строк

Изменения:

1. Исправлено зависание при нажатии кнопки «Оплатить» напротив счета.
   - Причина: после v11 UI polish несколько скриптов одновременно меняли текст кнопок оплаты, включая СБП, что могло вызывать конфликт обновлений.
   - Исправление: кнопки СБП исключены из автоматического переименования обычных кнопок оплаты, MutationObserver больше не реагирует на каждое изменение текста.

2. Восстановлено надежное открытие формы оплаты поверх страницы.
   - Кнопки «Оплатить» в счетах, документах и заказах снова открывают модальное окно оплаты.
   - Сумма и основание платежа подставляются в форму.

3. Добавлено заметное визуальное выделение выбранных карточек/строк.
   - Зеленая рамка.
   - Мягкая подсветка фона.
   - Зеленый левый индикатор выбранного элемента.

4. Сохранена логика СБП.
   - Кнопка «Оплатить по СБП» остается отдельной внутри формы оплаты.
   - Сумма в кнопке пересчитывается после списания бонусов/баланса.
