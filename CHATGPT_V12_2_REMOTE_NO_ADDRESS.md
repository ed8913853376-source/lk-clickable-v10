# v12.2 — удаленный доступ без адресов рабочих мест

Изменения:

- Убрано отображение AnyDesk/RuDesktop ID и адресов рабочих мест из интерфейса.
- Убрано копирование номера/адреса рабочего места.
- Не используются демонстрационные или вымышленные номера.
- В старом блоке быстрых действий остаются только кнопки AnyDesk/RuDesktop.
- Если внешний агент/backend передал `installed: true`, кнопка меняется на «Запустить ...».
- Если факт установки не передан, кнопка остается «Скачать ...».

Production-логика:

```js
window.lkRemoteAccess = {
  anydesk:   { installed: true, launchUrl: 'anydesk:' },
  rudesktop: { installed: true, launchUrl: 'edart-agent://launch/rudesktop' }
}
```

Без локального агента, расширения или backend-данных браузер не определяет установленную программу автоматически.
