# Исправление левой колонки v10.4

Проблема: в v10.3 верхние два блока левого меню не уменьшались, потому что сжатие было привязано только к scroll-событию `<nav>`. Если пользователь крутил колесико над карточкой клиента или блоком оплаты, событие не срабатывало.

Что изменено:

1. Добавлен обработчик `onWheel` на весь `aside`.
2. При прокрутке вниз сначала плавно увеличивается `sidebarProgress` от 0 до 1.
3. Верхний блок клиента скрывает все детали, оставляя только имя клиента.
4. Блок оплаты скрывает детали, оставляя кнопку `Пополнить / оплатить · 23 791 ₽`.
5. После полного сжатия меню продолжает обычную прокрутку.
6. При прокрутке вверх в начале меню блоки плавно раскрываются обратно.
7. Добавлен fallback на прокрутку основной страницы.

Проверка:

- `npm install` — успешно.
- `npm run build` — успешно.

Как проверить вручную:

1. Открыть `lk_client_portal_preview.html` или запустить `npm run dev`.
2. Навести мышь на левую колонку.
3. Крутить колесико вниз.
4. Сначала должны плавно сжаться верхние два блока.
5. В карточке клиента должно остаться только название клиента.
6. В блоке оплаты должна остаться только кнопка с суммой внутри.
