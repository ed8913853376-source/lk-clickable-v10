# FORMS_SPEC v10

Все формы открываются в модальных окнах. Все отправки показывают прототипный результат: `Заявка создана`, `Запрос отправлен`, `Заказ сформирован` или `Изменения отправлены на согласование`. Реальные API, 1С, Битрикс и платежи не подключаются.

## Добавить контрагента
Поля: Тип контрагента, Название организации, ИНН, КПП, ОГРН/ОГРНИП, Юридический адрес, Фактический адрес, Контактное лицо, Должность контактного лица, Телефон, Email для документов, Email для уведомлений, Комментарий.
Кнопки: Сохранить контрагента, Отмена.
Ограничение: нет полей базы 1С, сервера, лицензий или товара.

## Пользователь
Поля: ФИО, Email, Телефон, Должность, Контрагент, Роль, Доступ к разделам, Доступ к базам 1С, Комментарий.
Роли: Руководитель, Бухгалтер, Сотрудник, Технический специалист, Администратор.
Доступы: Финансы, Документы, Обращения, Заказы, Базы 1С, Серверы, Лицензии.
Кнопки: Сохранить пользователя, Отправить приглашение, Отмена.

## Новое обращение
Поля: Контрагент, Тип обращения, Связанный объект, Тема, Описание, Приоритет, Сколько пользователей затронуто, Удобный способ связи, Телефон, Файл/скриншот.
Типы: Ошибка 1С, Вопрос по работе 1С, Сервер/RDP, Лицензии, Оплата/документы, Заказ товара, Доставка, Другое.
Кнопки: Создать обращение, Отмена.

## База 1С
Поля: Контрагент, Название базы, Конфигурация, Способ создания, Пользователи с доступом, Нужна публикация, Нужно автообновление, Комментарий.
Конфигурации: Бухгалтерия, ЗУП, УНФ, УТ, ERP, Другая.
Способ создания: Новая пустая база, Загрузить .dt, Восстановить из backup, Скопировать существующую базу.
Ограничение: нет CPU, RAM, SSD, IP и тарифа сервера.

## Тонкий клиент 1С
Данные: Конфигурация, Версия платформы, Инструкция подключения, Список баз.
Кнопки: Скачать тонкий клиент 1С, Скачать инструкцию, Создать обращение, Закрыть.

## Изменение лицензий 1С
Поля: Контрагент, Текущий пакет лицензий, Используется сейчас, Свободно лицензий, Новое количество лицензий, Текущая стоимость, Новая стоимость, Разница, Комментарий.
Кнопки: Запросить изменение, Получить счет, Отмена.
Ограничение: нельзя уменьшить лицензии ниже текущего количества используемых.

## Сервер / ресурс
Поля: Контрагент, Тип ресурса, Название ресурса, Текущий тариф, Нужно изменить, Комментарий.
Типы: RDP, VPS, Сервер 1С, Файловый ресурс, Другое.
Кнопки: Отправить запрос, Создать обращение, Отмена.

## RDP-клиент с настройками
Данные: Сервер, Пользователь, Инструкция подключения.
Кнопки: Скачать RDP-файл, Скачать инструкцию, Создать обращение.

## Заказ услуги
Поля: Контрагент, Категория услуги, Описание задачи, Желаемый срок, Контактное лицо, Телефон, Email, Файл/ТЗ, Комментарий.
Категории: Сопровождение 1С, Доработка 1С, Интеграция 1С, Автоматизация бизнеса, Обслуживание касс, Серверы и аренда, Консультация, Другое.

## Заказ товара
Поля: Контрагент, Категория товара, Товар, Количество, Нужна доставка, Адрес доставки, Нужен монтаж/настройка, Нужны закрывающие документы, Комментарий.
Категории: Онлайн-кассы, Сканеры штрихкода, Принтеры этикеток, МФУ/принтеры, Расходные материалы, Комплектующие, Серверное оборудование, Рабочие места.

## Коммерческое предложение
Поля: Контрагент, Тип КП, Что нужно включить, Желаемый срок, Файл/спецификация, Комментарий.
Типы КП: Услуги, Товары, Смешанное КП.

## Заказ
Поля/данные: Номер заказа, Контрагент, Состав заказа, Статус, Сумма, Счет, Оплата, Адрес доставки, Транспортная компания, Трек-номер, Документы, Комментарий.
Статусы: Новый, Ожидает оплаты, В сборке, Готов к отгрузке, В пути, Доставлен, Закрыт.

## Финансы
Оплатить картой: Контрагент, Сумма, Назначение платежа, Комментарий.
Счет на оплату: Контрагент, Основание, Сумма, Email для отправки счета, Комментарий.
Привязать карту: Контрагент, Название карты, Email для чеков, Согласие на автосписание.
Обещанный платеж: Контрагент, Сумма, Причина, Дата планируемой оплаты.

## Документы
Поля: Контрагент, Тип документа, Период, Email для отправки, Комментарий.
Кнопки: Скачать, Отправить на email, Запросить акт сверки, Закрыть.

## Проект
Поля: Название проекта, Контрагент, Этап, Ответственный менеджер, Ответственный инженер, Сроки, Описание задачи, Последний комментарий, Файл/документ.
Кнопки: Запросить статус, Добавить комментарий, Принять этап, Создать обращение, Закрыть.

## Удаленный доступ
Текст: Для подключения специалиста скачайте программу удаленного доступа.
Кнопки: Скачать AnyDesk, Скачать RuDesktop, Инструкция по подключению, Создать обращение.


## Форма добавления контрагента по ИНН — v10.7

### Шаг 1. Ввод ИНН
Поля:
- ИНН

Кнопка:
- Найти / заполнить

### Шаг 2. Проверка найденных реквизитов
После ввода ИНН система прототипно заполняет:
- Тип контрагента
- Краткое название
- Полное юридическое наименование
- ИНН
- КПП
- ОГРН / ОГРНИП
- Налоговая система
- Юридический адрес
- Фактический адрес
- Банк
- БИК
- Расчетный счет
- Корреспондентский счет
- Контактное лицо
- Телефон
- Email для документов
- Email для уведомлений
- Комментарий

Кнопки:
- Сохранить контрагента
- Редактировать данные
- Отмена

Примечание: в реальном внедрении автозаполнение должно идти через 1С/CRM или внешний сервис подсказок.
