# v10.17 — правки раздела «Акты и документы»

Исправлено по замечаниям пользователя:

1. Добавлена кнопка «Заказать акт сверки».
2. По клику открывается форма с выбором контрагента и периода.
3. После формирования показывается готовый акт сверки и кнопка скачивания.
4. Список документов получил разную логику по типам:
   - счет на оплату: скачать счет, оплатить;
   - акт выполненных работ: скачать акт, скачать УПД, открыть счет-основание;
   - УПД: скачать УПД, скачать накладную;
   - акт сверки: скачать акт сверки или заказать новый;
   - договор / приложение: скачать или открыть договор.
5. Если клиент нажимает «Оплатить» у счета, сумма счета подставляется в верхний блок оплаты.
6. В верхнем блоке оплаты можно выбрать способ оплаты, скачать счет или документ.

Сборка проверена: `npm install`, `npm run build`.
