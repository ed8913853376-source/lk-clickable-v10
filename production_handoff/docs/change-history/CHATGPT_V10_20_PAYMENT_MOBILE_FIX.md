# v10.20 — исправление мобильной оплаты и компактных блоков

Исправлено по замечаниям:

1. Возвращена логика оплаты счета через основной платежный блок:
   - при нажатии «Оплатить» у счета сумма и основание передаются в форму оплаты;
   - сумма остается редактируемой;
   - основание заполняется номером заказа и счета.

2. Блок списания бонусов и баланса сделан компактнее:
   - описания на мобильной версии скрыты;
   - доступные суммы показываются рядом с названием;
   - поля списания появляются только после выбора чекбокса.

3. Мобильная версия раздела оплаты:
   - карточки «Баланс» и «Бонусы» остаются рядом в один ряд;
   - уменьшена высота карточек;
   - карточки «К оплате» и «Привязанные карты» на мобильной версии не показываются.

4. Burger-меню / левая панель:
   - при нажатии на «Счета и оплаты» на мобильной версии сразу открывается платежный блок;
   - кнопка «Пополнить / оплатить» в левом меню на мобильной версии также сразу переводит к платежной форме;
   - на компьютере поведение остается обычным: открывается раздел.

5. Дашборд-блоки на мобильной версии:
   - тяжелые счетчики dashboard скрываются через усиленное CSS-правило.
