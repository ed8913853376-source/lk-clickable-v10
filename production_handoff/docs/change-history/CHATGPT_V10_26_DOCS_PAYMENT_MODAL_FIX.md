# v10.26 — документы, оплата, дашборды

Изменения:
- В форме заказа акта сверки поля периода переведены в date input, чтобы была пиктограмма выбора даты из календаря.
- Кнопка «Оплатить» у счета в таблице теперь открывает поверх страницы отдельную форму оплаты.
- В форме оплаты сумма и основание подставляются из выбранного документа, сумму можно корректировать.
- Добавлен выбор основания платежа из созданных документов, остаток автоматически подставляется в сумму.
- Бонусы и баланс в форме оплаты стали компактнее: переключатели слева, сумма редактируется рядом.
- Верхние дашборды на главной странице получили клики: переходят в соответствующий раздел.
- Баланс и бонусы в разделе оплаты зафиксированы в один ряд.
