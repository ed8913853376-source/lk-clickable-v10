# Исправления v10.7 — контрагенты

1. В разделе «Контрагенты» карточки выровнены по высоте, а кнопки внизу всегда стоят в один ряд.
2. Кнопка «Добавить контрагента» открывает отдельный мастер добавления по ИНН.
3. Сценарий добавления:
   - пользователь вводит только ИНН;
   - нажимает «Найти / заполнить»;
   - прототип автоматически заполняет реквизиты организации;
   - пользователь видит найденные данные;
   - пользователь может дополнить и отредактировать поля;
   - после сохранения показывается прототипное состояние отправки на проверку.
4. Форма не содержит полей базы 1С, сервера, лицензий или товара.
5. Реальная загрузка данных по ИНН должна подключаться программистами через 1С/CRM или сервис подсказок.
