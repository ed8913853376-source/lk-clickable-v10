# Исправление добавления контрагента v10.8

## Что изменено

1. В форме «Добавить контрагента» добавлен переключатель типа:
   - Юридическое лицо / ИП
   - Физическое лицо

2. Для юридического лица / ИП сохранена прежняя логика:
   - сначала вводится ИНН;
   - затем нажимается «Найти / заполнить»;
   - реквизиты прототипно заполняются автоматически;
   - пользователь может проверить и отредактировать данные.

3. Для физического лица добавлен отдельный сценарий:
   - при выборе «Физическое лицо» прототип автоматически заполняет ФИО и телефон;
   - можно редактировать ФИО и телефон;
   - можно добавить несколько email;
   - можно добавить несколько адресов;
   - перед сохранением показывается блок «Что будет сохранено».

4. Не смешиваются поля юрлица и физлица:
   - для физлица не показываются КПП, ОГРН, банк, расчетный счет и другие реквизиты организации;
   - для юрлица не показываются динамические списки личных почт и адресов физлица.

## Для программистов

В реальной версии автозаполнение:
- по юрлицам/ИП — через 1С, CRM или сервис подсказок по ИНН;
- по физлицам — из профиля пользователя, CRM или вручную.
