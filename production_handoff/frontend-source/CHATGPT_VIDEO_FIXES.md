# CHATGPT_VIDEO_FIXES.md

Исправления по видео с озвучкой, версия v10.2.

## Что исправлено

1. **Модальные окна и длинные формы**
   - Длинные формы больше не уезжают за экран.
   - Заголовок и кнопки формы остаются на месте, содержимое формы прокручивается внутри окна.
   - Убрана тяжелая blur-подложка, чтобы интерфейс меньше тормозил.

2. **Форма добавления контрагента**
   - Пересобрана как форма реквизитов организации.
   - Добавлены банковские реквизиты: банк, БИК, расчетный счет, корр. счет.
   - Добавлены адреса и контактные данные.
   - В форме нет полей баз 1С, серверов, лицензий или товаров.

3. **Сертификат / промокод**
   - Кнопка применения кода теперь вызывает форму сертификата/промокода, а не универсальную форму.
   - Логика оставлена простой: одна строка для кода.

4. **Коммерческие предложения**
   - Кнопки КП подключены явно: открыть, согласовать, задать вопрос, скачать PDF.

5. **Левое меню и текущий пользователь**
   - Добавлен блок текущего пользователя с логином и кнопкой копирования.
   - Для мобильной версии добавлена кнопка закрытия меню внутри сайдбара.

6. **Пользователи**
   - Добавлена явная кнопка “Открыть” в таблице пользователей.
   - При выборе пользователя карточка справа обновляется понятнее.

7. **База знаний**
   - Карточки статей теперь открывают отдельное модальное окно статьи.
   - В статье есть кнопка копирования ссылки и создание обращения.

8. **Мобильная версия**
   - Уточнена ширина мобильного меню: не больше 86% экрана.
   - Добавлена кнопка закрытия меню.

## Проверка

- `npm install` выполнен успешно.
- `npm run build` выполнен успешно.

## Что требует ручной проверки

- Точные поля реквизитов контрагента: нужно подтвердить, нужны ли ЭДО, договор-основание, фактический/почтовый адрес отдельно.
- Реальные ссылки на AnyDesk, RuDesktop, тонкий клиент 1С и RDP-файлы программисты подключат при внедрении.

## v10.3 — плавное сворачивание верхних блоков левого меню

Исправлено по уточнению из видео:

- левый сайдбар теперь не прокручивает всю колонку целиком;
- прокручивается только навигационная часть меню;
- верхний блок клиента при прокрутке меню плавно уменьшается;
- в верхнем блоке в компактном состоянии остается только название клиента и статус;
- ИНН, менеджер, инженер, текущий пользователь и статус договора скрываются плавно;
- блок “Баланс и часы” при прокрутке меню плавно уменьшается;
- в компактном состоянии остается только кнопка “Пополнить / оплатить · 23 791 ₽”;
- сумма перенесена внутрь кнопки в компактном состоянии;
- строка “Остаток часов” и отдельный вывод суммы скрываются плавно;
- анимация сделана через opacity, max-height, transform и transition;
- визуальный скроллбар левого меню остается скрытым, прокрутка работает.
