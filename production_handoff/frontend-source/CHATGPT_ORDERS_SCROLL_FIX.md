# v10.11 — исправление прокрутки заказов и размеров шрифтов

## Что исправлено

1. Раздел «Заказы и отгрузки» теперь разбит на две независимые scroll-зоны:
   - левая колонка со списком заказов прокручивается отдельно;
   - правая колонка с детальной карточкой выбранного заказа прокручивается отдельно.

2. Колесо мыши работает по месту наведения:
   - если курсор над списком заказов, прокручивается только список заказов;
   - если курсор над карточкой «Заказы и отгрузки», прокручивается только детальная карточка заказа;
   - общий экран не должен тянуться вместе с этими колонками.

3. Добавлен `overscroll-contain`, чтобы скролл не передавался родительской странице при работе внутри колонок.

4. Уменьшены шрифты в компактных блоках заказа:
   - номер заказа;
   - дата;
   - трек-номер;
   - счет;
   - доставка;
   - контрагент;
   - менеджер.

5. Состав документа, таблица товаров/услуг и кнопки действий сохранены.
